Onboarding media
================

Drop a short GIF (or animated screen capture) here that shows how to get a
free Groq API key. The free onboarding step looks for:

    assets/onboarding/groq-key.gif

Suggested content (a few seconds, looping):
  1. Open https://console.groq.com/keys
  2. Sign in with Google
  3. Create API key -> copy

If the file is missing, the step gracefully hides the media panel and shows
the 3-step text + "Open Groq Console" button instead. Keep it reasonably
small (ideally under ~2 MB) so the modal stays snappy.
